# Aegis MTFI - v1.0 Release Notes

**June 2026**

---

## Interface & Workflow

- Replaced the previous long-scroll layout with a five-tab interface: Report, Formation, DNA Profile, Squad, Dashboard
- The written briefing report now generates automatically when an analysis completes - no button required
- A clean empty state with clear instructions replaces the previous blank screen on first load
- Formation compatibility is displayed as a subtitle beneath the manager and club names, giving an immediate at-a-glance reading before any tab is opened
- The comparison mode mirrors the same five-tab structure as single analysis, ensuring a consistent experience across both workflows

---

## Automated Briefing Reports

- Every analysis now produces a five-section written briefing: Executive Summary, Manager DNA Profile, Squad Impact Assessment, Recruitment Brief, and Risk & Transition Timeline
- Reports use plain football language - no statistical jargon - suitable for direct delivery to a Sporting Director or board
- Reports are viewable inline within the app and downloadable as a formatted PDF or HTML file in one click
- In Shortlist mode, each ranked manager has their own individual report and PDF download without switching modes
- Comparison mode generates a report for each scenario side by side

---

## Manager Shortlist Ranker

- A new Shortlist mode allows up to ten managers to be ranked simultaneously against a single squad in one run
- Results are displayed as a ranked table showing average fit score, tactical archetype, formation, Key Enablers, Marginalised players, and estimated recruitment cost per manager
- An overlaid DNA radar chart shows all shortlisted managers' profiles on a single axis for immediate side-by-side comparison
- Each manager in the shortlist has a drill-down view with full squad fit detail and a one-click PDF report
- This mirrors how clubs actually work - shortlists and ranked candidates, not one-off assessments

---

## Formation Analysis

- Formation data is now sourced live from StatsBomb match lineup records - replacing previously hardcoded placeholder values
- The platform analyses two distinct formation histories: the club's current shape based on their recent matches, and the incoming manager's preferred shape based on their tenure at their previous club
- A Formation Compatibility Score (0-100) compares the two shapes structurally, with a plain-English label: Identical, Compatible, Adaptable, or Divergent
- Automatic notes explain the key structural differences - for example, whether full-backs need to convert to wing-backs, or whether a midfield overhaul is required
- Two side-by-side formation history bar charts show how frequently each formation has been used, giving context beyond a single primary label
- Dual pitch diagrams show both the club's current shape and the manager's preferred shape with squad players positioned and colour-coded by classification
- A player transition risk table identifies which players change classification when the squad adapts to the manager's preferred formation
- Seven formations are fully supported with accurate position slot and pitch coordinate mapping: 4-3-3, 4-2-3-1, 4-4-2, 3-5-2, 3-4-3, 5-3-2, 4-1-4-1

---

## Manager DNA Insights

- A new DNA Insights panel surfaces four additional layers of intelligence alongside the radar chart
- Tactically Similar Managers identifies the closest tactical peers in the database by similarity percentage - for example, "This profile is 89% similar to Arne Slot"
- Pillar vs Archetype Benchmark compares each of the eight pillar scores against the average for that manager's tactical cluster, flagging where they stand out above or below their peer group
- A Confidence Indicator shows how many matches of StatsBomb data underpin the scores, banded as High, Medium, or Low
- The tactical archetype system now uses six distinct clusters - High-Press, Possession-Based, Counter-Attack, Defensive, Attacking, Balanced - trained across all available managers and seasons
- The manager pool for coach selection now draws from all four available seasons across all five licensed leagues, making historical managers accessible for hypothetical scenarios

---

## Squad Analysis

- The squad table now shows all squad members, not just the Ideal XI
- Player positions are shown in full detail - Left-Back, Right Winger, Defensive Midfield - rather than broad groups
- Classification filter buttons allow instant filtering to any group: Key Enablers, Good Fit, System Dependent, or Potentially Marginalised
- The Ideal XI Slot column shows which starting position each player has been assigned in the manager's system
- In comparison mode with the same club, a merged table shows each player's fit score under both managers with a delta column, highlighting who gains and who loses under each appointment
- Recruitment priorities are surfaced at the bottom of the Squad tab for immediate reference

---

## Player Dossier

- A dedicated Player Dossier mode generates a standalone scouting report for any player in the StatsBomb dataset
- Each dossier includes season statistics, percentile rankings across key metrics, a radar chart, and a scout rating summary
- Metrics are colour-coded by percentile to allow rapid assessment of strengths and weaknesses relative to peers
- Dossiers are available as downloadable HTML reports, suitable for sharing with recruitment staff or ownership groups
- The feature is accessible as a separate mode in the sidebar, independent of the main analysis workflow

---

## Analytical Model & Data

- A single master model is now trained across all five licensed leagues and all four available seasons, giving the clustering algorithm the richest possible population
- The model is pre-trained overnight via an automated daily process and committed to the platform - eliminating the 10-30 minute training wait that previously occurred on each run
- Cross-league analysis is now fully supported: any manager from any licensed competition can be assessed against any club, regardless of which league either party is in
- The league selector has been removed from the analysis inputs - the platform automatically detects the correct league from the selected club
- Formation values no longer silently default - if StatsBomb data is unavailable for a formation lookup, the platform displays a clear unavailable message rather than showing a misleading value
